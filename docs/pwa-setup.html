Away Mode (Beta)

When "Away Mode" is enabled, your Linked Lamp detects when your phone connects to the web dashboard. When you leave your lamp and it is tapped, you will receive a push notification natively on your iOS or Android device.

phonelink_ring Required Setup

Because Linked Lamp is 100% open-source and has no central backend server, you will need to create a free Firebase project to route push notifications directly to your phone.

1. Create a Firebase Project

Go to the Firebase Console and click "Add project". Enter a project name (e.g. "Linked Lamp"). Disable Google Analytics, and create the project.

2. Get Service Account Credentials (For the ESP32)

In your Firebase project, go to Project settings > Service accounts.

Click Generate new private key. This will download a JSON file. Open it in a text editor to find two things you need for the Software Setup Wizard:

  • The client_email starting with "firebase-adminsdk..."
  • The full private_key text block containing "-----BEGIN PRIVATE KEY-----"

3. Get Web Configuration (For your Phone)

In your Firebase project, go to Project settings > General. Scroll down and add a "Web app" () to your project. Register the app without hosting.

You will be presented with a configuration object that looks like this code block. Copy this text, because you will paste it into the Web App settings on your phone later.

const firebaseConfig = { apiKey: "AIzaSy...", authDomain: "linked-lamp.firebaseapp.com", projectId: "linked-lamp", storageBucket: "...", messagingSenderId: "123456789", appId: "1:123456789:web:abcdef" };

install_mobile Install Output Application

In order for background notifications to work effectively, you must save the web app to your home screen.

iOS Safari

1. Open your unique Web App URL link from the setup wizard in Safari.

2. Tap the Share icon (square with arrow pointing up) at the bottom.

3. Tap "Add to Home Screen".

Android Chrome

1. Open your unique Web App URL link from the setup wizard in Chrome.

2. Tap the three dots menu at the top right.

3. Tap "Install app" or "Add to Home screen".

Enable & Verify

Open the app from your home screen. Go to "Settings" and toggle the Away Mode switch. You will be prompted to paste the `firebaseConfig` object you copied in Step 3. After pasting, allow notification permissions when prompted.
