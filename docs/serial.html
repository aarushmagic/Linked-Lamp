terminal Command Reference

  • HELP Show all available commands
  • GET_STATUS Print WiFi, MQTT, and role status
  • GET_CONFIG Print the current /config.json
  • GET_STATE Print the current /state.json
  • SET_CONFIG:{} Set new MQTT config JSON and reboot
  • SET_WIFI:s,p Set new WiFi credentials and reboot
  • SCAN_WIFI Scan for nearby WiFi networks
  • RESET_WIFI Clear WiFi credentials and reboot (opens config portal)
  • RESET_CONFIG Delete MQTT config and reboot
  • MAKE_PRIMARY Promote to primary lamp (clears secondary status, reboots)
  • MAKE_SECONDARY Demote to secondary lamp (clears primary status, reboots)
  • RESET_ROLE Clear role and auto-detect on next boot
  • REBOOT Restart the device
  • RESET_ALL Factory reset — clear WiFi + config + state
