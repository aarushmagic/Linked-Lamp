Frequently Asked Questions

Everything you need to know about the Linked Lamp.

You'll need a few common components for each lamp:

1. An ESP32 development board (the brain).
2. Through-hole RGB Common Anode LEDs (diffused/frosted recommended, like Chanzon 5mm).
3. A TTP223 capacitive touch sensor module.
4. The custom Linked Lamp PCB (Gerber files provided in our setup guide).
5. A 3D-printed enclosure (STLs provided).
6. Some wire and a 5V USB power supply.

Because everything is open-source, you can source these parts yourself for just a few dollars each!

The Fix: In PlatformIO, you must run the "Upload File System image" task explicitly. This takes your `data/config.json` file and uploads it into the ESP32's LittleFS memory. Once that's done, restart the lamp, and it will connect instantly!

Because you are building and configuring your own lamp, updates are completely under your control via OTA (Over-The-Air) updates.

If we release a new feature or bug fix on our GitHub, you can navigate to your personalized Web App (`/my/` link), go to Settings, and tap "Check for Update". Your lamp will automatically download the latest firmware.bin from our public repository and restart itself—no plugging back into a computer required!

The ESP32 microcontroller only supports 2.4 GHz Wi-Fi networks. It will not connect to a 5 GHz-only network. Fortunately, almost all modern routers dual-broadcast both 2.4 GHz and 5 GHz bands, meaning as long as your router supports legacy devices, your Linked Lamp will connect perfectly fine.
