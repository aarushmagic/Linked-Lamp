WiFi Configuration Tool

Connect your lamp via USB to scan for networks, change WiFi settings, or troubleshoot connectivity issues.

Browser Not Supported

This tool requires Web Serial API, which is only available in Google Chrome or Microsoft Edge on desktop. Please switch to one of those browsers.
