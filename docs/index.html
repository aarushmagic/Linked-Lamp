Infinite Colors

Choose any color to represent your mood. Send a vibrant yellow for joy, or a peaceful blue for thinking of you.

language

Global Range

Powered by ultra-fast, secure MQTT brokers. If both lamps have Wi-Fi, distance doesn't matter.

app_shortcut

Powerful Web App

Control durations, night modes, colors, and save your favorite "signals" (presets) via a sleek web app—open on any device with a browser.

nights_stay

Smart Night Mode

Set a sleep schedule. Taps received during night mode will illuminate the lamp dimly (or keep it off) so you never get woken up.
